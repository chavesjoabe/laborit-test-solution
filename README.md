## Description

Application that transform a prompt in natural language into a SQL query

## Installation

```bash
$ npm install
```

## Running the app

### Environment variables
to execute the query and access the database is needed to create a .env file and add the following variables
```
TEXT_2_SQL_URL
TEXT_2_SQL_API_KEY
DB_TYPE
DB_HOST
DB_PORT
DB_USERNAME
DB_PASSWORD
DB_DATABASE
```
After set up your env vars the is ready to be executed

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Endpoints

The application exposes 2 endpoints that receives a prompt and process it using AI to generate a real SQL query.
We have 2 endpoints to process the prompts.  
> POST - /prompt/sql  

receives the prompt and return the SQL query to be used on database to get the response.  

> POST - /prompt/sql/response  

receives the prompt, generate query and perform the query on database and return the query result to the client.
